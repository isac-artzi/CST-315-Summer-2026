# Topic 5 — Processor Scheduling

Three subprojects:

- **`milestone/`** — `scheduler`: Round Robin simulator. Reads a workload, runs RR with a configurable quantum, prints a Gantt trace and metrics (waiting, turnaround, throughput, context switches).
- **`starter/`** — same shape with TODOs on queue ops and the scheduling decision.
- **`demos/`** — INSTRUCTOR ONLY. `fcfs_sim`, `sjf_sim`, `mlfq_sim`, `realtime_demo`.

The cumulative shell from Topic 3 is unchanged; `scheduler` is a standalone simulator the shell can invoke.
