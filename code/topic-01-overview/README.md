# Topic 1 — Overview of Computer Systems and Operating Systems

Three subprojects:

- **`milestone/`** — `hello` binary. The reference smoke-test program for Project 1.
- **`starter/`** — `hello.c` skeleton with TODOs. What students get.
- **`demos/`** — INSTRUCTOR-ONLY. `pipe_demo`, `env_demo`, `procfs_walker`, `whoami_clone`.

## What students should do this week

1. Stand up Linux/UNIX, install `gcc` and `git`.
2. Clone the student starter repo, `cd` into `topic-01-overview/starter/`.
3. Implement the TODO in `src/hello.c`.
4. `make && ./hello` to verify, `make test` to run the smoke test.
5. Push to their private GitHub.
