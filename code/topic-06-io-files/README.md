# Topic 6 — I/O and File Management

Three subprojects:

- **`milestone/`** — `fs`: an in-memory hierarchical file system with a REPL. Supports the core commands required by Project 6 (mkdir, rmdir, touch, rm, ls, tree, cd, pwd, info, find, mv, cp). Builds the foundation; Project 6 extends it with detailed-info flags and additional flags.
- **`starter/`** — same shape with TODOs on path parsing, recursive removal, and tree traversal.
- **`demos/`** — INSTRUCTOR ONLY. `stat_demo`, `fd_table_demo`, `simple_raid0`, `mmap_io`.

The cumulative shell from Topic 3 is unchanged; `fs` is a standalone REPL the shell can invoke.
